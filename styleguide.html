Typography

Primary typeface - Open Sans

Use Open Sans as the default typeface for the website. It should be used for both headings and body text.

Example Heading
Example paragraph: Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed maximus vel dolor in pretium. Proin vitae semper erat. Ut vitae ultrices neque. Aliquam tristique ultricies condimentum. Etiam felis lacus, feugiat at ligula vel, eleifend convallis turpis. Aenean tempor augue eget varius gravida. Nam vel augue sed ante porttitor pulvinar. Pellentesque enim ex, elementum quis sodales vitae, pulvinar ac ante.

Secondary typeface - Source Sans Pro

Source Sans Pro should be used primarily for setting long-form, running text on screens. Avoid using weights other than dsaf Regular. Use Source Sans Pro Bold for emphasis when needed.
